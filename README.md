### Installation



```sh
# making an virtual environment

python3 -m venv my_ve
source my_ve/bin/activate

cd neuron_dash # directory that have setup.py
pip3 install -e .
```


#### prerequisite:

```
wheel
dash-bootstrap-components
numpy
scipy
dash
pandas
```
